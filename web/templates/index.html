    Power On Open Hand Spread all 5 fingers open to turn the lamp ON
    Power Off Closed Fist Curl all fingers into a fist — lamp OFF & camera sleeps
    Toggle Smart Thumbs Up Thumb pointing up, 4 fingers curled — toggles Smart Auto-Brightness
    Brightness Pinch Thumb & index pinch together — spread wider to increase brightness
    Pan Lamp Peace / Point Index + middle (or just index) extended — move left/right to rotate lamp
    Auto Sleep Inactivity Timeout No gesture for 60 seconds → lamp off, camera released automatically
    💡 Smart Mode note: Pinch brightness control only works when Smart Auto-Brightness is OFF. Use Thumbs Up to toggle it. When Smart Mode is active, the LDR ambient sensor adjusts brightness automatically. Manual pinch overrides it for 10 seconds before the sensor resumes.
    🎙️ How to start: Say "activate gesture", "start camera", or "wake" — or press 🖐️ Activate Gestures in the Lamp Control card. Gestures only work when the camera is active.
    • Lamp Button (Click): Cycle brightness (20% → 100% → 20%)
    • Lamp Button (Hold): Turn Lamp ON / OFF
    • Rotate Button (Click): Rotate right 20°
    • Rotate Button (Hold): Continuous smooth pan left
    • Hold Both Buttons (1.5s): Enter Document Scan Mode
    • Lamp Button (in Scan Mode): Capture current page
    • Focus Tracker: Uses camera face tracking. Triggers visual alerts if distracted (looking away).
    • Pomodoro Timer: Follows 25-minute study, 5-minute break intervals to maximize productivity.
